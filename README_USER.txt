Spotify Playlist Manager v1.3.0 - エンドユーザー向け配布パッケージ

インストール手順：

1. SpotifyPlaylistManager_v1.3.0.zip を任意のフォルダに展開してください
2. 展開されたフォルダ内の SpotifyManager.exe をダブルクリックして起動します

注意事項：

・このアプリケーションはインストール不要のポータブル版です
・フォルダ全体をそのまま保持してください（すべてのファイルが必要です）
・Windows Defender等でブロックされた場合は、信頼できるアプリとして許可してください
・初回起動時にSpotifyアカウントでのログインが必要です

システム要件：

・Windows 10/11 (64bit)
・.NET 8.0 Runtime（自動的にインストールされます）
・インターネット接続（Spotify API用）

サポート：

・アプリケーションの機能については、同梱されたREADME.mdをご確認ください
・問題が発生した場合は、GitHubリポジトリにてお問い合わせください

ファイル内容：

・SpotifyManager.exe - メインアプリケーション
・各種DLLファイル - 実行に必要なライブラリ
・appsettings.json - 設定ファイル
・icon.ico - アプリケーションアイコン
・言語リソースフォルダ（cs, de, es, fr, it, ja, ko, pl, pt-BR, ru, tr, zh-Hans, zh-Hant）
